# 🎵 Музыкальный анализ личности

Веб-приложение для анализа личности на основе музыкальных предпочтений через реляционную базу данных PostgreSQL.

## 🎯 Что делает проект

1. Пользователь вводит 20 любимых треков
2. OpenRouter API (Llama 3) анализирует треки → возвращает energy/valence
3. Данные сохраняются в PostgreSQL на Railway
4. SQL-запрос анализирует распределение по 4 зонам
5. Выдается один из 7 типов личности

---

## 🗄️ База данных

**4 таблицы:**
- `users` — пользователи (имя, возраст, пол)
- `tracks` — треки с характеристиками (energy, valence)
- `playlists` — плейлисты пользователей
- `playlist_tracks` — связь многие-ко-многим

**Текущие данные:**
- 8 участников
- 160 треков
- Хостинг: Railway (PostgreSQL)

---

## 🧠 Типы личности

1. **Эйфорический экстраверт** — энергичная позитивная музыка
2. **Тревожный активист** — высокая энергия, низкий позитив
3. **Меланхолик** — тихая грустная музыка
4. **Созерцательный оптимист** — спокойная позитивная музыка
5. **Меланхолик с агрессией** — смесь грусти и злости
6. **Позитивный микс** — разнообразная позитивная музыка
7. **Эмоционально нестабильный** — равномерное распределение
